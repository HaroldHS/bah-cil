## Glossary
* *Abstract Syntax Tree*, or *AST*, object that represent logical structure of a language
* *Intermediate Representation*, or *IR*, syntax between raw text and machine code 
* *Lexer*, or *Tokenizer*, tool that scans or consumes a given input and returns the token type 
* *Non-terminal Symbols*, variables or placeholders representing syntatic categories that can be substituted with other symbols
* *Parser*, tool that scans or consumes a given input based on BNF & scanned tokens from Lexer and returns an AST
* *Terminal Symbols*, basic symbols that make up strings of a language & can't be broken down furthermore
